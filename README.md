# nichenet-learning-reproduction

This repository contains my learning-based reproduction of a NicheNet workflow using R, Seurat, and NicheNet.

## Files
- `scripts/00_install_packages.R`: install and load packages
- `scripts/01_prepare_data.R`: download and prepare data
- `scripts/02_run_nichenet_analysis.R`: main NicheNet analysis
- `scripts/03_plot_results.R`: result visualization

## Notes
This is a learning-oriented reproduction project for practice and understanding of the workflow.
